# Security Policy

## Reporting a vulnerability

If you believe you have found a security vulnerability, please do not open a public issue.

Instead, report it privately:

- Create a private message to the repository owner on GitLab (preferred), or
- Email the maintainer if contact details are available on the project page.

Please include:

- A description of the issue
- Steps to reproduce
- Potential impact

## Supported versions

Security fixes will be applied to the `main` branch.
